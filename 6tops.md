## Quick Start

- **Factory Boot**
  - MaskROM
  - Boot from eMMC

### M.2 B-Key (4G)

- **Link peripheral**

  First, connect 4G Module to board .
  You can check if the device is connected by the following command:
  ```
  $ lsusb | grep -i Quectel
  Bus 001 Device 003: ID 2c7c:0125 Quectel Wireless Solutions Co., Ltd. EC25 LTE modem
  ```
